Structure of a Webpage

HTML (HyperText Markup Language) is the standard markup language used to create the structure and content of web pages. A well-formed HTML document always follows a specific foundational structure that web browsers understand and use to render the page correctly.

DOCTYPE

Every HTML 5 document starts with a DOCTYPE html declaration. This is not an HTML tag; it is an instruction to the web browser about what version of HTML the page is written in, ensuring the browser renders it in standard mode.
